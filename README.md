MyBatis Aggregator
==================

![mybatis-logo](http://mybatis.github.io/images/mybatis-logo.png)

MyBatis-Aggregator is the MyBatis aggregator POM which builds all MyBatis modules.

## Running ##

Use these JVM settings

-Xmx512m -XX:MaxPermSize=128m
